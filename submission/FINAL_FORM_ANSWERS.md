# TON Compass Final Form Answers

## Project name

TON Compass

## One-line description

TON Compass is a guided onboarding layer for first-time TON swaps that uses the official STON.fi widget for execution and adds a Journey Builder, a readiness preflight, a live STON.fi quote and wallet-aware decision layer, a Tonstakers-ready next step, and a subscription-ready Telegram continuation layer.

## Short description

TON Compass helps new TON users make their first DeFi move without getting dropped into a raw trading interface. Instead of showing every option at once, it frames the first action through guided routes, a user-profile-based Journey Builder, an interactive readiness layer before wallet connection, and a live STON.fi-powered swap panel. The current version also supports shareable route URLs, a Decision Lab that pulls official STON.fi simulations and wallet-aware hints from the public API, a Tonstakers continuation planner, and a Compass Pro preview for Telegram-based follow-up alerts and saved routes, so the product feels like the beginning of a TON journey rather than a single embedded widget.

## Problem

For a new TON user, DeFi often feels like too much too early. Wallet connection, gas, slippage, token selection, and swap mechanics all appear at once, which creates friction at the exact moment when trust matters most.

## Solution

TON Compass narrows the first interaction into guided routes. The Journey Builder maps the user profile to the right route, the readiness layer slows the user down before opening the wallet, and the live STON.fi widget handles the real swap execution. The app now also includes a Decision Lab that uses official STON.fi public endpoints for route simulation and wallet-aware hints, then offers a clear Tonstakers continuation instead of stopping at the transaction itself, plus a Telegram-based continuity layer for saved routes, alerts, and future subscription follow-up.

## Why this project should be selected

- It is a new project being built specifically for this hackathon

- It uses the official STON.fi widget for real execution, not a mock transaction flow
- It already demonstrates how a small STON.fi app can evolve into a broader TON journey product with a credible monetization path
- We are ready to keep building through the official hackathon window and submit the full package on time

## Current functionality

- Journey Builder that recommends the best first route from goal, experience, and TON balance
- Three guided routes: ecosystem entry, open market mode, learn before swap
- Interactive readiness / preflight layer before wallet connection
- Live STON.fi swap execution
- Live STON.fi route simulation layer
- Wallet-aware recommendation layer based on official STON.fi wallet asset endpoints
- Shareable route URLs with persistent mode state
- Tonstakers continuation planner with live TON ↔ tsTON market signals
- Compass Pro preview layer for Telegram continuity, saved routes, and subscription-ready follow-up
