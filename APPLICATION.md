# TON Compass Application Notes

## Project name

TON Compass

## Short description

TON Compass is a beginner-friendly first-swap app for TON. It gives new users one clear entry point into DeFi, simple context in plain language, and a real execution path through the official STON.fi widget.

## Problem

Most DeFi products assume users already understand wallets, slippage, token selection, and swap flows. For a new TON user, that creates too much friction at the very first interaction.

## Solution

TON Compass wraps the first swap in a guided experience. Instead of overwhelming the user with a raw terminal-like interface, it offers simple presets, pre-flight guidance, and a live STON.fi-powered execution layer.

## Why this project should be accepted

- It solves a real onboarding problem for TON users
- It already has a working public demo
- It uses a real STON.fi integration instead of a fake on-chain mock
- It is small enough to finish fast, but structured enough to keep growing after the hackathon

## Current status
