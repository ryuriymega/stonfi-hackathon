# TON Compass Submission Draft

## One-line summary

TON Compass is a beginner-friendly TON DeFi entry point that wraps the official STON.fi swap flow in a much simpler onboarding experience.

## Problem

For a new TON user, most DeFi interfaces feel like too much too early. The first interaction should feel guided and trustworthy, not overloaded with options.

## Solution

TON Compass reduces the first action to a few guided presets and a plain-language explanation of what the user is about to do. Instead of rebuilding swap logic from scratch, it embeds the official STON.fi Omniston Widget for the actual execution layer.

## Why it fits the STON.fi track

- Real integration with the official STON.fi widget
- Clear use case instead of a generic demo page
- Fast to ship and easy to evolve into a larger onboarding product

## Current functionality

- Guided landing page with a product narrative
- Three onboarding modes:
  - Ecosystem entry
  - Open market mode
  - Learn-before-swap mode
- Dynamic TON Connect manifest generated by the app endpoint
- Live STON.fi widget mount through the official CDN script
- Running publicly inside the existing Node-RED instance on this server

## What makes it credible

- The app is already structured as a standalone product, not a code snippet
- The execution layer is real, not mocked
- The architecture is minimal and maintainable enough to keep building on this server

## Next evolution after the hackathon

1. Add beginner-friendly explanations for gas, slippage, and asset risk
2. Track common first-swap journeys with lightweight analytics
3. Add a Tonstakers mode for "stake first, then explore DeFi"
4. Personalize presets based on user intent, such as stablecoins, ecosystem tokens, or learning mode
