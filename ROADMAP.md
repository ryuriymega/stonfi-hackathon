# TON Compass Roadmap

- Guided first-swap landing page
- Real STON.fi integration

- Glossary for new users

- Better wallet-state handling and transaction feedback

## Phase 3: Product expansion

- Add a Tonstakers starter flow
- Add user journey analytics
- Add reusable onboarding widgets for other TON apps
- Add a simple admin config for featured token pairs and educational content
